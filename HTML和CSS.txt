。1.实体（转义字符）
	-在HTML中不能直接输入一些特殊字符，往往需要通过实体进行转义
	-实体的语法：&实体名字；（后面加分号）
		-大于号：&gt;
		-小于号：&lt;
		-空格：&nbsp;
		-版权声明：&copy；
2.相对路径
	-相对路径就是相当于当前页面所在的路径
	-使用../返回一级目录，返回几级写几个../
3.图片格式的选择：
	-web中常见的图片格式有：JPEG、GIF、PNG
	-JPEG:支持的颜色比较多，可以显示鲜艳的图片，一般情况下照片、颜色复杂的图片都会用。（不支持透明）
	-GIF：支持的颜色比较少，可以显示一些颜色单一的图片，比如网站单色的LOGO。（支持简单的透明，支持全透明，不支持半透明）
	-PNG：支持的颜色比较多，很牛逼。（支持全透明）
	选用原则：显示效果一致，用图片小的
			  显示效果不一致，使用效果合适的 
	  
4.超链接a
	-属性：
		-href:要跳转到的目标地址
		-target:可以用来指定在哪个位置打开目标页面,
				可选值：
						_self:默认值，在当前页面打开
						_blank:在新的页面打开目标页面
						内联框架的名字：将会在指定的内联框架打开目标页面
	在a标签中，href=“#”中“#”表示自动回到当前页面的顶部
	当运用a标签到指定位置时，需要给指定位置设置id属性，这是运用<a href=“#id属性值”>就可以到设置id属性的位置
5.CSS选择器
	5.1复合选择器（交集选择器）
		-作用：可以选择符合多个要求的元素
		-语法：选择器1选择器2选择器N...
		-例如：div.hello  (表示既是div又是class名为hello的元素，中间没有任何符号)
	5.2子元素选择器
		-作用：选中指定父元素中的指定子元素
		-语法：父元素>子元素
	5.3后代元素选择器
		-作用：可以选中指定元素的指定后代元素
		-语法：祖先元素 后代元素
	5.4.兄弟元素选择器
	10.1	语法：兄弟元素1+兄弟元素2
	作用：选择兄弟元素1后一个兄弟元素2
	10.2	语法：兄弟元素1~兄弟元素
	作用：选择兄弟元素1后面所有的兄弟元素

		-访问过的链接      ：visited
		-鼠标移入的链接    : hover
		-链接被点击        ：active

8.属性选择器
	title id class 都可以
	[属性名] 选择具有指定属性名的元素
	[属性名=属性值] 选择属性名等于指定值的元素
	[属性名^=属性值] 选中属性名以指定值开头的元素

	[属性名*=属性值] 选中属性名中包含指定值的元素
	

9.样式的优先级
	内联样式 				优先级1000
	ID选择器的样式			优先级100
	类选择器的样式			优先级10
	元素和伪类选择器的样式	优先级1
	通配选择器的样式		优先级0
	继承来的样式 			没有优先级
10.字体的样式
	-font-size 设置字体大小
	-font-weight 设置字体是否加粗 
		属性值：bold normal
	-font-family 设置字体的格式
	-font-style 设置字体是否为斜体
		属性值：normal italic
	-font-variant 设置文字是否是小型大写字母
		属性值 small-caps normal
11.文本样式
	-line-height  为文本设置行高
		-行间距=行高-字体大小
		-如果想要为单行文本设置垂直居中时，只需要将高度和行高设置为相同的值，则该单行文本会自动垂直居中
	-text-transform  可以用来设置内容的大小写
		-none，默认值，正常显示
				capitalize，首字母大写，通过空格来判断是否是一个单词
				uppercase，将所有的文本设置为大写
			    lowercase，将所有的文本设置为小写
12.外边距问题（margin）
	   1当外边距左右，上下设置外边距冲突时，优先显示上外边距和左外边距
	2外边距可以为负值，当值为负值时会往相反的方向移动
	3我们经常使一个元素的左右外边距设置为auto,以使其在父元素居中
	4外边距重叠问题,相邻元素的垂直外边距会发生重叠现象
13，display的none和visibility的hidden都会使元素隐藏，
		其中dispaly:none 元素会直接消失，并且原来元素的位置不会占用，
		而visibility：hidden 元素会消失，但是元素原来的位置依然占用
14.overflow
	可以指定父元素如何处理子元素
	可选值：
		-visible 默认值，溢出内容依然会在元素外显示
		-hidden  溢出的内容会隐藏
		-scroll  为父元素添加滚动条，无论实际情况需要不需要都会添加
		-auto    网页会自动为父元素添加滚动条，根据实际需要添加，不需要就不显示
15.浮动
	1.元素设置浮动以后，会尽量向上浮动，或左或右，但是它最高不会超出父元素上边框
	2.如果浮动元素的上边是一个没有浮动的块元素，则它不会超过该块元素
	3.浮动元素不会超过他上边的浮动的兄弟元素，最多最多一边齐
	4.块元素浮动以后，不会再独占一行
	5.浮动元素不会覆盖文字，文字会环绕在浮动元素的周围，我们可以使用这个特点来完成文字环绕图片的效果，(这才是设置浮动属性的初衷)
16.清除浮动(高度塌陷问题)
	高度塌陷问题：父元素包含的子元素浮动起来之后，父元素的高度就不会再被子元素所撑开，这时父元素的高度会丢失
	每一个元素都有一个隐藏属性，那就是BFC属性。解决高度塌陷问题可以通过开启BFC(块级格式化上下文)来解决
	BFC开启后，会具有以下属性：
	1.开启BFC属性后，子元素的垂直外边距不会传递给父元素
	2.开启BFC属性后，不会被浮动元素所覆盖
	3.开启BFC属性的元素可以包含浮动的子元素(它的高度可以被浮动子元素撑开)
	开启BFC可以用overflow：hidden，这条语句也可以不完美的解决高度塌陷问题
	
	当我们为了解决一个高度塌陷的元素时，我们可以给该元素添加一个空的子元素div，并且为该元素设置clear：both；代码，清除这个空元素上面的浮动元素对自己的影响，从而
	让这个空的div撑起来整个父元素，在显示效果上就清除了浮动

	完美解决高度塌陷问题：
	clearfix:before,clearfix:after{
		content:"";
		disable:table;
		clear:both;     
	}
17.opacity属性
		-可以用来设置元素的透明背景
		-它需要一个0-1之间的值
			0表示全透明
			1表示完全不透明
18.z-index
	-当元素开启了定位后，可以通过z-index来设置元素的层级
	-如果z-index值一样，则后边的元素会优先与前边的元素显示
	-父元素的层级再高，它也不会盖住它的子元素
19.背景(background)
	1.background-attachment
		-设置背景图片是否随滚动条滚动
		-属性值：
				-scroll 默认值  背景图片会随滚动条滚动
				-fixed  图片不随滚动条滚动
	2.background-position
		语法：background-position：x偏移量；y偏移量
		x偏移量:指定背景图片的水平的偏移量
					- 如果指定一个正值，则背景图片会向右移动指定的像素
					- 如果指定一个负值，则背景图片会向左移动指定的像素
				y偏移量:指定背景图片的垂直偏移量
					- 如果指定一个正值，则背景图片会向下移动指定的像素
					- 如果指定一个负值，则背景图片会向上移动指定的像素

IE6的兼容问题
1.hover和active在IE6中不支持为其他元素绑定，只支持为超链接设置
2.IE6不支持子元素选择器
3.在IE6中，超过父元素的子元素，会将父元素撑开，以使其可以包含子元素
4.在IE6中，width和height实际上不是固定的宽度和高度，而是一个最小值
5.IE6不支持overflow属性，所以用overflow：hiddde,不能清除浮动（高度塌陷问题）。
	可以用zoom：1来开启IE6的HasLayout来解决该问题
	也可以通过为父元素设置一个非默认值的数值也会开启haslayout,从而解决高度塌陷问题
6.IE6不支持固定定位
7.IEB以下的浏览器不支持opacity这个属性
	在IE8以下中用filter:alpha(opacity=透明度)
	0表示全透明；100表示完全不透明。
8.forEach不支持IE8及以下的浏览器，所以如果需要兼容IE8 forEach就不要用了。
